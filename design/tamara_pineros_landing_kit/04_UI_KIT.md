# 04 — UI Kit y lineamientos visuales

## 1. Dirección visual
Base visual tomada de las referencias entregadas:
- fondo gris muy claro con matiz lila,
- contraste alto en tipografía,
- acento violeta sobrio,
- monograma simple,
- composición limpia,
- mucho aire,
- jerarquía fuerte en titulares.

## 2. Personalidad visual
- moderna
- jurídica sin verse “clásica”
- premium
- sobria
- tecnológica ligera
- directa

## 3. Paleta recomendada

### Fondo principal
`#F8F8FB`

### Fondo secundario / superficies
`#F1F0FB`

### Texto principal
`#23262E`

### Texto secundario
`#53627A`

### Acento primario
`#5A4FCF`

### Acento hover
`#4E43C4`

### Borde suave
`#D9DCE5`

### Fondo oscuro opcional para bloques premium
`#2D313C`

## 4. Gradientes permitidos
Usar gradientes muy sutiles:
- gris claro a lila humo,
- oscuro azul gris a violeta profundo.

Nunca usar:
- neones,
- degradados saturados,
- efecto gamer,
- brillos exagerados.

## 5. Tipografía
### Recomendación principal
- **Manrope** para titulares
- **Inter** para cuerpo y UI

### Alternativa de una sola familia
- **Manrope** en todo el sitio

## 6. Escala tipográfica sugerida
### Desktop
- H1: 64 / 1.0–1.05
- H2: 40 / 1.1
- H3: 26 / 1.2
- body lg: 22 / 1.5
- body: 18 / 1.6
- small: 14 / 1.5

### Mobile
- H1: 44
- H2: 30
- H3: 22
- body: 17
- small: 14

## 7. Botones
### Primario
- fondo violeta
- texto blanco
- border radius alto (pill)
- padding generoso
- sombra muy ligera

### Secundario
- borde gris suave
- fondo transparente o muy claro
- texto oscuro

## 8. Tarjetas
- radio entre 20 y 28 px
- borde 1 px suave
- sombra baja
- mucho padding interno
- iconografía mínima

## 9. Espaciado
Usar respiración amplia.

### Secciones desktop
- 96 a 128 px vertical

### Secciones mobile
- 64 a 80 px vertical

### Contenedor
- max-width: 1200 px
- padding lateral: 24 px mobile / 40 px desktop

## 10. Imágenes y recursos
- priorizar monograma,
- no saturar con fotos stock genéricas de abogados,
- si se usan personas: retrato limpio, real, neutro, elegante,
- si no hay fotos de calidad, usar layout tipográfico fuerte con shapes suaves.

## 11. Estilo de íconos
- lineales,
- simples,
- sin volumen,
- sin degradados complejos.

## 12. Sensación general
La página debe sentirse como:
**“un servicio jurídico confiable, actual y fácil de entender.”**

# Matrices operativas TP

## 1. Matriz de servicios

| Servicio | Entrada | Entregable | Canal |
|---|---|---|---|
| Revisión de arriendo | Contrato + aviso incremento | Respuesta práctica / orientación | WhatsApp + PDF |
| Asesoría express | Consulta concreta | Ruta de acción | Videollamada / WhatsApp |
| Constitución de empresa | Datos del negocio | Ruta + documentos base | Formulario + llamada |
| Derecho de petición | Hechos + entidad | Documento revisado/redactado | Drive + WhatsApp |
| Tutela | Hechos + pruebas | Revisión de viabilidad / documento | Drive + llamada |
| Conciliación | Conflicto + contraparte | Estrategia + solicitud | WhatsApp + carpeta |
| Empresa local | Necesidad recurrente | Paquete mensual | Llamada + seguimiento |

## 2. Matriz de roles

| Rol | Función principal | No debe hacer |
|---|---|---|
| CEO / fundador | Visión, alianzas, servicios, control general | Atender todo manualmente para siempre. |
| Director jurídico | Criterio legal, revisión, calidad | Manejar ventas sin sistema. |
| Coordinador operativo | Ordenar leads, carpetas, tareas y tiempos | Dar conceptos jurídicos sin autorización. |
| Abogado aliado | Ejecutar servicios asignados | Contactar clientes por fuera del sistema. |
| Responsable comercial | Contenido, WhatsApp, seguimiento comercial | Prometer resultados jurídicos. |
| Asistente documental | Drive, plantillas, archivos, versiones | Manipular estrategia jurídica. |

## 3. Matriz de indicadores

| Área | Indicador | Meta inicial razonable |
|---|---|---:|
| Contenido | Retención de video | Mejorar video a video. |
| Redes | Comentarios con palabra clave | Medir intención. |
| WhatsApp | Tiempo primera respuesta | Menos de 2 horas hábiles. |
| Comercial | Leads calificados / leads totales | 30–50% inicial. |
| Ventas | Conversión a producto pagado | 10–20% inicial. |
| Operación | Tiempo de entrega | Según servicio. |
| Calidad | Retrabajos por error | Menos de 10%. |
| Cliente | Referidos / reseñas | Medición mensual. |

## 4. Matriz de riesgos comerciales

| Riesgo | Control |
|---|---|
| Muchos leads curiosos y pocos clientes | Lead magnets con filtro y CTA a producto pagado. |
| Servicios demasiado amplios | Productizar: entrada, alcance, precio, entregable. |
| WhatsApp desordenado | Etiquetas, scripts y responsable diario. |
| Marca bonita pero sin ventas | Medir clics, leads calificados y conversión. |

## 5. Matriz de riesgos operativos

| Riesgo | Control |
|---|---|
| Abogados aliados trabajan cada uno a su manera | Manual operativo y plantillas obligatorias. |
| Documentos perdidos | Drive con estructura fija. |
| Versiones confundidas | Regla de nombres y control de versiones. |
| Cliente sin seguimiento | Tablero con estado obligatorio. |
| Sobrecarga del fundador | Delegar intake, documentos y seguimiento. |

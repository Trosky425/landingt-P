# Flujos operativos TP

## 1. Flujo general de atención

```text
1. Usuario ve contenido
   ↓
2. Comenta / escribe por WhatsApp
   ↓
3. Mensaje automático de bienvenida
   ↓
4. Clasificación del asunto
   ↓
5. Solicitud mínima de información
   ↓
6. Revisión de conflicto de interés
   ↓
7. Propuesta de servicio / honorarios
   ↓
8. Pago o aceptación
   ↓
9. Creación de carpeta Drive
   ↓
10. Asignación a abogado
   ↓
11. Ejecución
   ↓
12. Revisión interna
   ↓
13. Entrega al cliente
   ↓
14. Seguimiento
   ↓
15. Cierre, reseña o referido
```

## 2. Flujo comercial

```text
Contenido → Comentario → WhatsApp → Clasificación → Oferta → Pago → Caso activo
```

## 3. Flujo documental

```text
Lead calificado → Intake → Carpeta Drive → Documentos cliente → Borrador → Revisión → Versión final → Entrega → Cierre
```

## 4. Flujo del pool de abogados

```text
Caso activo → Área jurídica → Abogado asignado → Plazo → Ejecución → Revisión de calidad → Entrega autorizada
```

## 5. Flujo de cierre

```text
Entrega → Confirmación cliente → Soporte en Drive → Cierre tablero → Solicitar reseña/referido → Métrica
```

## 6. Mensaje automático inicial WhatsApp

```text
Hola. Gracias por escribir a Tamara & Piñeros Abogados Asociados.

Para orientarte mejor, cuéntanos brevemente:

1. ¿Cuál es tu nombre?
2. ¿En qué ciudad estás?
3. ¿Tu consulta es sobre arriendo, empresa, tutela, conciliación u otro tema?
4. Resume tu caso en máximo 5 líneas.

Con esa información revisamos cuál puede ser el siguiente paso.
```

## 7. Etiquetas WhatsApp

- Nuevo lead
- Pendiente datos
- Calificado
- Consulta pagada
- En gestión
- Pendiente pago
- Cerrado
- Referido
- No viable
